إِنَّ أَوَّلَ بَيْتٍ وُضِعَ لِلنَّاسِ لَلَّذِي بِبَكَّةَ مُبَارَكًا وَهُدًى لِلْعَالَمِينَ
### meaning: 
Indeed the first house to be set up for mankind is the one at Bakkah, blessed and a guidance for all nations.
